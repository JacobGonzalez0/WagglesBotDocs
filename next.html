Next Command

This command gets the next image from the previous derpi search in the channel, if available.

Next can be used in conjunction with both Derpibooru search types, which return 50 images. If ~n runs through all images in search, it will start over. Images are filtered based on channel, see whitelist for more information.

Note: All derpibooru commands are subject to a whitelist set up manually, which is channel specific, see whitelist for more information.

Aliases

~n

Usage:

~next
~next <index>

Description:

Returns next image from the last derpibooru search made in this channel.

Index:

Allows you to reference a specific image within the search. For example, if the last search has 50 images in memory, running ~n 23 will return the 23rd image in the set
