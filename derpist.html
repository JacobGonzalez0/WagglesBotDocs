Derpist Command

This command gets the amount of matching images for a given Derpibooru query. Single tags, and a combination of tags both work.

Images are filtered based on channel, see whitelist for more information.

Aliases

~count, ~st, ~amount

Usage:

~next
~derpist <query>

Description:

Returns the amount of images on Derpibooru that match those tags.

Query:

Any query that would normally be put into a Derpibooru search. Foalcon is not included in the total results.
This search will only return the amount of images that are currently allowed through the filter. NSFW results are only counted if channel is nsfw.
