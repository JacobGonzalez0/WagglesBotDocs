Whitelist

The whitelist is an important part of Waggles functions, the majority of her Derpibooru commands use it to execute. It is, as its name suggests, a whitelist of channels that are NSFW enabled. Adding a channel to the whitelist allows for commands run within the channel to run without content restrictions.
Waggles will assume all channels are sfw by default, changing this is very simple. Note: Regardless of whitelist status, anything against Discord TOS is also prevented from showing up. This includes within result counts, and collection numbers.
If a channel is not within the whitelist, as is default for all channels that Waggles sees, commands will still execute normally. Only safe results/number counts are displayed.

Commands

~wlist add
~wlist remove
~wlist list

Usage:

~wlist all
~wlist <action> <#channel>

Description:

This command will add/remove the provided channel to the whitelist.

Channel:

Any channel placed within this argument will be added or removed from the whitelist. #channel format is preferred.
Using the argument all will loop through each channel visible to Waggles and add them to the whitelist. This is useful for allowing NSFW searches server-wide.
