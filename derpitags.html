DerpiTags Command

This command returns the tag for a given image, the artist and content rating being put forward first.
Tags of an explicit image will only be returned in a nsfw channel

Images are filtered based on channel, see whitelist for more information.

Aliases

~dt, ~tags

Usage:

~derpitags
~derpitags <link>

Description:

Returns all tags for provided image! If link is not provided, Waggles will return the tags of the last image posted in the channel.

Link:

When a link is provided Waggles will return the tags, any derpibooru link should work.
